* {Installation} Instructions

 The installation of Firemox consists solely of unpacking the archive in the desired directory. It's that simple!

** Windows 2000/XP

  [[1]] Unzip <<<magic-project_bin-0.85-bin.zip>>> to the directory you wish to install Magic Project. These instructions
        assume you chose <<<C:\Program Files\Magic-Project>>>

  [[2]] Double-click on <<</usr/local/magic-project/magic.bat>>> file.
  
** Unix-based Operating Systems (Linux, Solaris and Mac OS X)

  [[1]] Extract the distribution archive to the directory you wish to install Magic-Project. These instructions
        assume you chose <<</usr/local/magic-project>>>. The directory <<<magic-project>>> will be created from
        the archive.

  [[2]] Make sure the <<</usr/local/magic-project/magic>>> command has the <<< x >>> flag using the <<<chmode>> command.
  
  [[3]] Execute the <<</usr/local/magic-project/magic>>> command.  
  
** All

  [[4]] During the first execution, Magic-Project will try to get some pictures from Internet like mana symboles, card pictures,...
  
  [[5]] Once finished, Magic-Project is installed, and you can configure it like you want.
  
  
For more information, visit the site : http://magic-project.sourceforge.net
